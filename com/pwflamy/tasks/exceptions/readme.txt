Представим, что ты создаёшь небольшую программу для интернет-магазина, где пользователь может купить товар,
указав его название и количество.

1. Создай два собственных исключения:
ProductNotFoundException — проверяемое исключение (checked). Выбрасывается, если запрошенный товар не найден.
OutOfStockException — непроверяемое исключение (unchecked). Выбрасывается, если товара не хватает на складе.

2. Создай класс Product:
Поля: String name, int stock. Начальное значение их устанавливается в конструкторе.

Добавь публичный метод с именем "buy", который не возвращает ничего, а принимает в качестве аргумента количество товара "quantity".
Если quantity > stock — выброси OutOfStockException, иначе уменьши stock на quantity.

3. Создай класс Store:
Поле: список товаров (можно List<Product>), устанавливаются с помощью конструктора.
Добавь приватный метод "findProduct", который будет возвращать Product из списка по переданному в метод аргументу "String name".
Если товар не найден — выброси ProductNotFoundException. Обрабатывать исключения в этом методе не нужно.

Добавь публичный метод "purchase", который принимает в качестве аргументов "name" продукта и его количество "quantity".
Реализация: Найди товар с помощью метода "findProduct" и вызови у него метод buy(quantity).

Оберни всё в try-catch, обработай оба типа исключений.

Если все прошло успешно, выведи сообщение: "Успешно куплен товар (name) в количестве (quantity) штук".
Если товар по имени не найден: "Товар с именем (name) не найден".
Если запрашиваемое количество превышает количество на складе: "Недостаточно товара (name). Текущее количество - (текущее количество)"
Вне зависимости от успешности выполнения выведи сообщение: "Спасибо за визит!"

4. Создай класс Main:
Добавь несколько товаров в магазин.
